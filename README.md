# Responsive Landing Page

## Overview

This project is a fully responsive landing page with an interactive fixed navigation menu. The navigation bar changes its background color when the user scrolls down the page, and menu items feature hover effects for a modern user experience. The design is optimized for both desktop and mobile devices.

## Features

* Fixed Navigation Bar
* Navbar Background Change on Scroll
* Interactive Hover Effects on Menu Items
* Full-Screen Hero Section
* Responsive Design for Mobile and Desktop
* Smooth CSS Transitions
* Modern UI Design
* Background Image with Overlay Effect

## How It Works

1. The navigation menu remains fixed at the top of the page.
2. When the user scrolls more than 50px, JavaScript adds the `scrolled` class to the navbar.
3. The navbar background changes from transparent to dark.
4. Menu items display a highlight effect when hovered.
5. The layout automatically adjusts for smaller screens using CSS media queries.
